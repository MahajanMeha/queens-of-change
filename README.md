# Queens Of Change Foundation 👑

A modern and responsive NGO landing page developed for the Queens Of Change Foundation Internship Selection Task.

## 🌟 Features
- Responsive design
- Modern UI/UX
- Hero section with tagline
- About section
- Initiative cards
- Volunteer / Join Us button
- Dark mode toggle
- Mobile-friendly layout

## 🛠️ Tech Stack
- React
- TypeScript
- CSS
- Vite

## 🚀 Run the Project

```bash
npm install
npm run dev
